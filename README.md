# 📺 YouTubeタイムスタンプ整形ツール

YouTubeの概要欄（説明欄）に貼る「チャプター（目次）」を、手持ちのメモから一瞬で作成できるブラウザ完結型のツールです。

表記のゆれを自動修正したり、編集でズレた時間を一括補正する機能を搭載しています。

## ✨ 特徴

- **📝 どんなメモでもOK**
  - `1:30 本題`、`5分45秒 まとめ`、`10.15 おまけ` など、バラバラな形式を貼り付けるだけで統一されたフォーマットに整形します。
- **⏱️ 時間ズレの一括補正**
  - 「オープニングを10秒追加した」等の場合に、全てのタイムスタンプを `+10秒` `−5秒` のように一括計算して修正できます。
- **✅ YouTube最適化**
  - チャプター有効化に必須な `00:00` を自動挿入。
  - 1桁の分秒を `01:05` のようにゼロ埋め整形。
  - 60分を超える動画（`65:00`）を `01:05:00` へ自動変換。
- **📐 豊富なオプション**
  - 区切り文字の変更（スペース、ハイフン、縦線）。
  - 各チャプターの「尺（長さ）」を自動計算して表示。
  - YouTube動画リンク（`t=秒数`）の生成。
- **🚀 インストール不要**
  - HTMLファイル1つで動作。サーバーも通信も不要で、オフラインでも使えます。

## 使い方

1. `timestamp-tool.html` をブラウザ（Chrome, Edge, Safariなど）で開きます。
2. 左側の入力エリアに、タイムスタンプのメモを貼り付けます。
3. 必要に応じて「設定・補正」パネルで以下の操作を行います。
   - 区切り文字のスタイル変更
   - 時間のズレ補正（秒数入力）
   - 尺の表示オン/オフ
4. 右側に整形されたテキストが表示されるので、「コピーする」ボタンを押してYouTubeの概要欄に貼り付けます。

## 🛠️ 技術スタック

- **HTML5**
- **JavaScript (Vanilla JS)** - 外部ライブラリ依存なしの軽量設計
- **Tailwind CSS (CDN)** - モダンでレスポンシブなUIデザイン

## 📦 インストール方法

特別な環境構築は不要です。

1. このリポジトリの `timestamp-tool.html` をダウンロードします。
2. ファイルをダブルクリックして開くだけで使用可能です。
